# 💱 Currency Converter API (.NET + MySQL + Docker)

A containerized backend API built with **.NET 10**, **Entity Framework Core**, and **MySQL**, designed for currency conversion and exchange rate management. The project demonstrates clean architecture, database migrations, and full Docker support.

---

## 🚀 Features

- ⚡ ASP.NET Core Web API (.NET 10)
- 🐬 MySQL database integration
- 🧠 Entity Framework Core (Code First)
- 🔄 Automatic database migrations on startup
- 🐳 Fully Dockerized setup (API + MySQL)
- 📦 RESTful API architecture
- 🛡 Retry mechanism for DB connection stability
- 🧩 Clean separation of concerns (Controllers, Services, Data Layer)

---

## 🏗️ Tech Stack

- **Backend:** ASP.NET Core Web API (.NET 10)
- **ORM:** Entity Framework Core
- **Database:** MySQL 8
- **Containerization:** Docker & Docker Compose
- **Language:** C#
